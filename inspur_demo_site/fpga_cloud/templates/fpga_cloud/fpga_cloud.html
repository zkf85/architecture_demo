Overview

FPGA Cloud is a project aiming to enable communication between FPGAs directly. Traditionally, when a system operating with multiple FPGA, the workload is always organized and operated by CPU, including sending and receiving data to/from each FPGA board. The new FPGA Cloud technique presented by Inspur enables the "direct" talk between FPGAs through the network instead of CPU. Thus part of the workload that oringally on CPU can be unloaded to FPGA. Moreover, the latency of a large computing task on multiple FPGAs would be reduce significantly as the amount of FPGA boards grows.
